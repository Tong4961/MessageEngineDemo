# MessageEngineDemo

Spring Boot ж¶ҲжҒҜеј•ж“ҺзҪ‘е…іпјҢз»ҹдёҖжҺҘж”¶ HTTP / SOAP иҜ·жұӮпјҢиҪ¬еҸ‘иҮі Apache RocketMQпјҢе№¶ж”ҜжҢҒ RPC еҗҢжӯҘеӣһи°ғгҖӮ

## жҠҖжңҜж Ҳ

- Spring Boot
- Apache RocketMQ 5.xпјҲPush Consumerпјү
- Apache CXFпјҲJAX-WS SOAPпјү
- CompletableFutureпјҲRPC еҗҢжӯҘпјү

## жөҒзЁӢеј•ж“Һ

BP жЁЎеқ—еҶ…зҪ®жөҒзЁӢеј•ж“ҺпјҢж”ҜжҢҒй…ҚзҪ®еҢ–жөҒзЁӢеӨ„зҗҶпјҡ

- `ProcessConfig`пјҡжөҒзЁӢй…ҚзҪ®пјҲеҢ…еҗ«иҠӮзӮ№еҲ—иЎЁгҖҒиө·е§Ӣ/з»“жқҹиҠӮзӮ№пјү
- `ProcessNode`пјҡиҠӮзӮ№е®ҡд№үпјҲnodeIdгҖҒnodeTypeгҖҒnextNodeIdпјү
- `ProcessEngine`пјҡжөҒзЁӢжү§иЎҢеҷЁпјҢйҒҚеҺҶиҠӮзӮ№й“ҫ

еҶ…зҪ®иҠӮзӮ№зұ»еһӢпјҡ`START`гҖҒ`TEST`гҖҒ`END`
